## hangul-virtual-keyboard
기존의 simple-keyboard에서 제공하는 Korean 버전은 받침이 입력되지 않는 문제점이 있습니다. 이를 해결하기 위해 hangul-js의 Hangul.assemble을 이용했습니다. 한글의 자음/모음을 조합하여 글자로 보여줍니다.

* `npm start`
